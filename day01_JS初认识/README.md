## 变量 ##

来做一个简单的计算，那么必须要知道数据是如何存储的


### 标识(zhi)符 ###

由 字母(a-z, A-Z)、数字、下划线、$ 组成的式子，称为标识符。 
必须是以字母、下划线、$ 开头， 不能以数字开头

示例:

abc 是的
a_123 是的
长度 不是
___ 是的
$len 是
2x 不是


1. 组成方式
    字母、数字、下划线、$ 
    不能以数字开头

2. 建议
    (1) 不能重名
	(2) 名字尽可能有意义
	(3) 单词之间首字母尽量大写  datalength -> dataLength
